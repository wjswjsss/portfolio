# Assets

Place your media here:

- `cv.pdf` — your CV (linked from the Home page "Download CV" button).
- `projects/` — project GIFs / images referenced in `js/data.js`, e.g.:
  - `projects/robot-arm.gif`
  - `projects/detection.gif`
  - `projects/webapp.png`

Recommended:

- Use compressed GIFs (or short MP4s) to keep load times low.
- Project images look best at a 16:9 aspect ratio.
- System architecture diagrams can be exported as PNG and used the same way.

If an image is missing, the card simply hides its media area — no broken images.
